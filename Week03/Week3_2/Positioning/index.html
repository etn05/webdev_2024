Relative
Absolute positioned divs look for a parent element with a definted position property to position against. If it can't find one, it positions absolutely to the body tag.
Fixed behaves similarly to absolute but will be fixed on the screen regardless of scrolling.

When we have a relative parent, absolute or fixed position child elements will position in relation to the relatively positioned parent

fixed position same relative parent
